CUFC-UI Library is a collection of React components that are used in CUFC-UI projects.

## Installation

```bash
npm install @cufc/ui
```

## Usage

```jsx
import React from 'react'
import { Button } from '@cufc/ui'

const App = () => {
  return (
    <BaseButton
      text="Click me"
      onClick={() => console.log('clicked')}
    />
  )
}
```
